# Caesar Cipher Tool (Prodigy Infotech - Task 1)

A Python program that implements the Caesar Cipher algorithm to encrypt and decrypt text messages using a user-defined shift value.
This project was developed as part of the Prodigy Infotech Cyber Security Internship.

## 📌 Task

Create a Python program that can encrypt and decrypt text using the Caesar Cipher algorithm. The program allows users to enter a message and a shift value to perform encryption and decryption.

## 🚀 Features

* Encrypt text using the Caesar Cipher algorithm
* Decrypt encrypted text
* Supports uppercase and lowercase letters
* Preserves spaces, numbers, and special characters
* User-defined shift value
* Simple command-line interface

## 🛠️ Technology Used

* Python 3

## 🔍 How It Works

The Caesar Cipher is a substitution cipher in which each letter is shifted by a fixed number of positions in the alphabet.

Example (Shift = 3):

A → D
B → E
C → F

X → A
Y → B
Z → C
