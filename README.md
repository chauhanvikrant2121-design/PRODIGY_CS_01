# Caesar Cipher Tool (Prodigy Infotech - Task 1)

This is a Python-based implementation of the Caesar Cipher algorithm, developed as part of my internship task for **Prodigy Infotech**. The program allows users to encrypt and decrypt text messages by providing a custom shift value.

## 🚀 Features
* **Encryption:** Converts plain text into cipher text using a specified key shift.
* **Decryption:** Decodes cipher text back into plain text using the correct key shift.
* **Input Validation:** Safely handles numeric shift entries and preserves non-alphabetic characters (spaces, punctuation).

## 🛠️ Tech Stack
* **Language:** Python 3.x
